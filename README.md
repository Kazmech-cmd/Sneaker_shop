# Интернет-магазин кроссовки «SNEAKERS»

Курсовой проект: Веб-приложение представляет собой функциональный интернет-магазин с динамическим каталогом, интерактивной корзиной и автоматизированной системой уведомлений о заказах через Telegram API.

Разработано как демонстрация навыков построения клиент-серверной архитектуры (PHP + MySQL) без использования сторонних фреймворков.


Технологический стек

Backend: Чистый PHP, сессии для хранения состояния корзины, архитектура PDO для безопасных подготовленных запросов к БД.
Frontend: Чистый JavaScript (Fetch API для асинхронной отправки данных), HTML5, CSS3 (адаптивная верстка Flexbox/Grid) без использования сторонних фреймворков и медиа-ресурсов.
База данных: MySQL.
Интеграции: Telegram Bot API (архитектура Client-side request).


Основные возможности

Динамический каталог: Вывод моделей кроссовок, доступных размеров и цен напрямую из базы данных MySQL.
Умная корзина: Добавление товаров, выбор размеров, удаление позиций с мгновенным автоматическим пересчетом общей суммы заказа.
Уведомления в Telegram: При оформлении заказа администратор мгновенно получает детальную карточку заказа (Имя клиента, телефон, состав корзины, итоговый чек) в реальном времени.
